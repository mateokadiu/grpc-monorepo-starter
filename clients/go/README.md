Generated by `buf generate` from `proto/orders/v1/orders.proto`. The

## Regenerate

```bash
buf generate          # from the repo root
```

This drops fresh files under `clients/go/pb/orders/v1/` using
`buf.build/protocolbuffers/go` and `buf.build/grpc/go` remote plugins.
You do not need a local Go protoc-gen install — Buf fetches the plugins.

## Use

```go
import (
    "context"

    "google.golang.org/grpc"
    "google.golang.org/grpc/credentials/insecure"

    ordersv1 "github.com/mateokadiu/grpc-monorepo-starter/clients/go/pb/orders/v1"
)

func main() {
    conn, _ := grpc.NewClient("localhost:50051", grpc.WithTransportCredentials(insecure.NewCredentials()))
    defer conn.Close()
    client := ordersv1.NewOrdersServiceClient(conn)
    order, _ := client.GetOrder(context.Background(), &ordersv1.GetOrderRequest{Id: "ord_x"})
    _ = order
}
```
