# Python client — `orders/v1`

Generated by `buf generate` from `proto/orders/v1/orders.proto`. The
committed files (`orders/v1/orders_pb2.py`, `orders_pb2_grpc.py`) are
stubs included for the 60-second `degit` experience — regenerate via Buf
when extending the proto.

## Regenerate

```bash
buf generate          # from the repo root
```

This drops fresh files under `clients/python/orders/v1/` using
`buf.build/protocolbuffers/python` and `buf.build/grpc/python` remote
plugins.

## Install

This tree is not pip-installable as-is; copy the `orders/` package
into your application or vendor via:

```bash
pip install grpcio grpcio-tools protobuf
PYTHONPATH=clients/python python -m my_app
```

## Use

```python
import grpc
from orders.v1 import orders_pb2, orders_pb2_grpc

with grpc.insecure_channel("localhost:50051") as channel:
    stub = orders_pb2_grpc.OrdersServiceStub(channel)
    order = stub.GetOrder(orders_pb2.GetOrderRequest(id="ord_x"))
    for ord_msg in stub.ListOrders(orders_pb2.ListOrdersRequest(customer_id="cus_1")):
        print(ord_msg.id)
```
